# Site Mapper

A collection of tools and services to facilitate mapping of large gov website

## Usage

### CLI mode

`python3 Start.py www.domain.gov`

### POLL mode
`python3 Start.py`

domain will be obtained from a [webservice](http://openciti.ca/cgi-bin/peek)
